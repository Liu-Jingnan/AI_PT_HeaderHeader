# 概率论与数理统计学习辅助工具

这是一个基于人工智能的概率论与数理统计学习辅助工具，旨在帮助学生更好地理解概率论与数理统计的概念、解决相关问题、生成练习题并检查答案。

## 功能特点

1. **概念解释**：详细解释概率论与数理统计中的各种定义和概念
2. **例题讲解**：分析概率论与数理统计问题并给出详细解答
3. **生成练习题**：根据指定主题创建涵盖核心概念的练习题
4. **检查答案**：检查概率论与数理统计练习题的正确性并提供反馈

## 环境配置

### 1. 安装依赖

```bash
pip install python-dotenv openai
```

### 2. 配置API密钥

在项目根目录下创建`.env`文件，并添加以下内容：

```env
OPENAI_API_KEY=your_api_key_here
```

**注意**：本项目使用的是智谱AI的GLM-4模型，需要配置相应的API密钥。

## 使用方法

1. 运行主程序：

```bash
python PY_AI_Header.py
```

2. 根据提示选择任务类型：

```
请选择任务类型（输入对应数字）：
1. 概念解释
2. 例题讲解
3. 生成练习题
4. 检查答案 
5. 退出
```

3. 输入相关内容，等待AI导师的响应

## 技术实现

- **AI模型**：使用智谱AI的GLM-4模型
- **API客户端**：OpenAI Python客户端库
- **环境管理**：python-dotenv库管理环境变量
- **流式输出**：支持AI响应的实时流式展示
- **记忆管理**：自动清理超出限制的对话历史

## 注意事项

1. 请确保已正确配置API密钥，否则程序无法正常运行
2. 本工具仅作为学习辅助，不能完全替代专业的教材和老师
3. 网络连接不稳定可能导致API请求失败，程序会自动重试最多5次
4. 对话历史会自动限制在最近10轮，以避免超出API的上下文限制
